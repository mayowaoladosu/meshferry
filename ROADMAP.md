# Roadmap

## Phase 1

- Stable HTTP tunnel flow
- Better CLI ergonomics
- Docker publishing
- Request logging and metrics
- Authentication hardening

## Phase 2

- HTTPS and TLS termination
- TCP and WebSocket passthrough
- Persistent tunnel registry
- Access policies and rate limits
- Custom domains

## Phase 3

- Multi-region edge support
- Team and project management
- Audit logs
- Managed control plane
- API gateway policies
